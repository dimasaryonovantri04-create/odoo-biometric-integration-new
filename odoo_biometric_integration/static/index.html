Logging & Audit Trail

Setiap data mentah dari mesin dicatat dalam log khusus sebelum diproses, memudahkan pelacakan dan audit.

Pemrosesan Real-Time

Data yang masuk langsung diproses menjadi catatan absensi (`hr.attendance`) tanpa jeda.

Antarmuka Monitoring

UI khusus untuk melihat status data (Pending, Processed, Error) dengan kode warna yang informatif.

Proses Ulang Manual

Admin dapat memproses ulang data yang gagal (error) langsung dari UI, memberikan kontrol penuh.
