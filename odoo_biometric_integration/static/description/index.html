  1. Unduh modul dan letakkan di dalam direktori addons Odoo Anda.
  2. Restart server Odoo.
  3. Aktifkan Developer Mode, lalu update daftar aplikasi (`Apps > Update Apps List`).
  4. Cari modul ini (`Odoo Pro Biometric Integration`) dan klik "Install".
  5. Navigasi ke menu `Attendances > Configuration > Settings` untuk mengisi Secret Key dan Timezone agar modul dapat berjalan.
